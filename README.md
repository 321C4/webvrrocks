# WebVR Rocks

Your one-stop resource for everything about WebVR.

**[WebVR Rocks!](https://WebVR.rocks)**


## Installation

```sh
npm install
```


## Local development

To start the local dev server:

```sh
npm start
```

Built using [jus](http://jus.js.org/).


## Deployment

To deploy the static pages to production:

```sh
npm run deploy
```
